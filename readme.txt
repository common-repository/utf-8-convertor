This plugin is design to convert all the characters that appear on your blog posts to UTF-8 English text only

== Description ==

This plugin is designed to convert all the broken UTF8 characters on your WordPress powered site.
It will leave all your posts intact in the database and instead convert the ugly characters each time a post is loaded 


== Installation ==

1. Download the installation archive and uncompress
2. Upload the unarchived folder to your plugins directory (this is wp-content/plugins/)
3. Activate the plugin in your WordPress administration panel (this is the Plugins page)
